### 使用hexo搭建博客

#### 安装 hexo-cli -> hexo init object  ->  npm i -> hexo s 启动服务
#### hexo new "文章名" 生成文章
#### hexo g 生成文件
#### hexo d 部署 （之前先安装部署插件 如github：npm install hexo-deployer-git --save）
#### 部署目标在配置文件中设置(必须给git设置有户名邮件 和验证ssh git的仓库名是 git名.github.io)
